Reinforcement learing using SARSA(lambda) Algorithm

Grid World

Similarly to the ANN project, there are plenty of libraries that handle the hard work of reinforcement learning for you. However, in order to fully appricate and understand how reinforcement learning works, I opted to implement the logic behind the sarsa(lambda) algorithm from scratch in a javascript application. With the object set to explore 20% of the time and exploit 80% of the time and the decay rate set to .5, the object learned to navigate the grid in about 6 days. The process would have been much quicker if I had not tied the calculations to run at the same speed as the GUI (using javascript phaser.io game engine). Moving forward, I plan to reimplement this project in a C++ project and compare the results. You can see a demo of the project running below.
